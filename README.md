Welcome to aysCloud!

aysCloud is an Electron application.

It is powered by aysTab, but slightly different.

PROS

    It has its own branding
    You always have the newest aysCloud version
    Usually, Scratch projects don't remember you. Not with aysCloud, though!

CONS

    aysCloud requires an internet connection. This is because it needs to download the latest version.
    If you reset your PC, aysCloud won't remember you. This is because your AppData folder also gets wiped.
    aysCloud eats a lot of RAM and storage. This is because aysCloud is just Chrome.
