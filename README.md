Welcome to aysOS!

aysOS is an Electron application.

It is powered by regular aysOS, but is slightly different.

PROS

1. It has its own branding
2. You always have the newest aysOS12 version
3. Usually, Scratch projects don't remember you. Not with aysOS, though!

CONS

1. aysOS requires an internet connection. This is because it needs to download the latest version.
2. If you reset your PC, aysOS won't remember you. This is because your AppData folder also gets wiped.
3. aysOS eats a lot of RAM and storage. This is because aysOS is just Chrome.
